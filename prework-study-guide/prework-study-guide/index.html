HTML

    • Some more information about the head element;
      a number of meta elements (elements that are hidden
      from the webpage separate from what is shown in the body element)
      they tell the browser information about data this can be things such,
      identifying the character set for the webpage or describing the content
      in a way that helps search engines understand what it is the webpage is trying to accomplish
      and lastly, it can also allow the page to become responsive.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Both the head and body elements are refered to as semantic elements meaning that it makes our webpage more easily read by screenreaders and search engines.

CSS

  • CSS is a language that is defined by a selector to identify which element the style changes are to be attached to and a code block stating which changes should be made.
  • A basic syntax for CSS would look like this.
    p {
    font-family: Helvetica;
    }

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • When staging i.e saving files this important to have all modifications for the project updated into your branch prior to being pushed to Github.
    • git add -A
  • We must commit our changes to the repository (repo for short) so that we can prepare to push our changes that we have made to our remote repo from our local repo.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A for loop is an automatically called piece of code that is called on the loading of the webpage
  • A function is a piece of reusable code that has to be called before it can be used.
